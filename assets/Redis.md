# Redis

#### 服务端启动:

#### 	redis-server.exe redis.windows.conf

##### 	  ctrl+c退出

#### 客户端启动:

#### 	redis-cli.exe -h localhost -p 6379 -a 2023

######                                                                        -h     服务端ip   -p 端口号    -a  密码

##### 	 exit退出



#### 密码设置:

#### redis.windows.conf: requirepass+密码
